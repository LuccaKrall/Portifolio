-Intuito do Projeto

Este projeto marcou o início do meu aprendizado em Java. Após meu primeiro contato com a linguagem, comecei a explorar o Java Swing para a criação de interfaces gráficas e decidi desenvolver um projeto prático para consolidar meu aprendizado. Como tenho interesse na área de investimentos, resolvi criar um banco virtual, com funcionamento local. Inicialmente, por falta de conhecimento sobre bancos de dados, optei por utilizar arquivos de texto (.txt) como forma de armazenamento. Estudei e implementei um sistema que salva e gerencia informações de maneira local, utilizando arquivos como base para registrar os dados do banco.

Esse projeto foi fundamental para meu desenvolvimento, permitindo-me compreender melhor conceitos de programação, manipulação de arquivos e construção de interfaces gráficas. Além disso, ele reforçou minha capacidade de resolver problemas de forma criativa, mesmo com recursos limitados.

-Oque o aplicativo faz:

O aplicativo que desenvolvi é uma simulação de um banco virtual voltado para investimentos. O funcionamento inicia com uma tela de registro, onde o usuário pode criar um login, definir uma senha e registrar sua renda inicial. Esse valor será utilizado dentro do aplicativo para realizar compras de ações.

Ao fazer login, o usuário tem acesso a diversas funcionalidades financeiras, incluindo:

Pagamentos:É possível realizar pagamentos por meio de boleto, Pix ou cartão de crédito. Cada usuário possui um número exclusivo de cartão e uma chave secreta para garantir segurança.

Compra de ações:Na área de investimentos, o usuário pode selecionar a quantidade de ações desejada e proceder com o pagamento.

Após realizar as compras, o usuário pode acessar uma área dedicada para visualizar suas ações adquiridas. Nessa seção, é possível filtrar por diferentes categorias, facilitando o gerenciamento dos investimentos. Esse projeto simula um ecossistema financeiro completo, permitindo que os usuários explorem conceitos como transações, gestão de recursos e investimentos, tudo em um ambiente virtual seguro e dinâmico.
