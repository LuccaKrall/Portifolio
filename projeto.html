Projeto Sistema de Agendamento para Clinica Odontológica

Sobre o Projeto:

-Tecnologias Utilizadas

-Intuito do Projeto

Este projeto foi desenvolvido com o objetivo principal de aprimorar meu aprendizado em Java, uma linguagem de programação que comecei a estudar durante a faculdade e com a qual me identifiquei. Além disso, busquei aplicar esse conhecimento para resolver problemas reais. Meu irmão possui uma clínica odontológica e enfrentava dificuldades com a organização do dia a dia, especialmente na gestão da agenda de pacientes. A partir dessa necessidade, decidi criar um aplicativo que pudesse solucionar os principais problemas enfrentados, combinando aprendizado técnico com a resolução de desafios práticos.

-Oque o aplicativo faz:

O projeto consiste em um aplicativo desenvolvido para atender as necessidades de uma clínica odontológica, com foco principal no cadastro e agendamento de pacientes. O sistema conta com dois tipos de login, um para funcionários e outro para dentistas, cada um com permissões específicas:

Login de Funcionários:Permite acessar funcionalidades como cadastro de pacientes, agendamento e gerenciamento completo dos dados dos pacientes (CRUD).

Login de Dentistas:Além de oferecer as mesmas funcionalidades disponíveis para os funcionários, concede acesso total ao sistema, incluindo o gerenciamento de funcionários e dentistas. O dentista também tem acesso a uma agenda personalizada, que exibe compromissos do dia, do próximo dia e do mês.

O sistema inclui ainda uma área de atendimento para registrar observações sobre os pacientes, que são salvas no banco de dados para consultas futuras. Essa funcionalidade contribui para um acompanhamento mais preciso e eficiente do histórico do paciente. Esse projeto reflete uma solução robusta e prática para o gerenciamento de uma clínica odontológica, integrando organização e eficiência no dia a dia da operação.
